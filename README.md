# nine-docs

Source for the [Nine Engineering Hub](https://nine-2.gitbook.io/nine-docs/) — the central knowledge base for engineering practices, platform documentation, runbooks, and architectural decisions across Nine Digital.

Each top-level folder is a separate GitBook space connected via Git Sync.
